- Proof of Stake: validators deposit 32 ETH, get pseudorandomly selected to propose blocks
- RANDAO selects the proposer for each 12-second "slot"
- If the selected proposer is offline, the slot stays empty — small inactivity penalty, network continues
- [[NPhAUEqa|Attestation]] committees vote on block validity — >2/3 needed for finality
