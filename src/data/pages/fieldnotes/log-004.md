# markdown reference

this log documents every markdown pattern supported by the build system.

## text formatting

regular paragraph text flows naturally. you can use **bold text** for emphasis, *italic text* for subtle emphasis, and ~~strikethrough~~ for corrections or deprecations. combine them for ***bold and italic*** when needed.

## lists

unordered list:
- first item
- second item
- third item with more detail

ordered list:
1. step one
2. step two
3. step three

## links

inline links work like this: [infraphysics home](https://infraphysics.net) or reference external resources like [github](https://github.com).

## code

inline code uses backticks: `const x = 42;`

code blocks use triple backticks:

```javascript
function entropy(system) {
  return system.disorder / system.order;
}
```

```python
def measure_friction(task_a, task_b):
    return abs(task_a.context - task_b.context)
```
