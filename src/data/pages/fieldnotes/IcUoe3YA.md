- Piezoelectric effect: applying voltage to quartz crystal makes it vibrate at exactly 32,768 Hz
- Provides the timing reference every [[MTfcKkH5|chip]] needs to synchronize operations
- Critical for radio protocols; Bluetooth must emit at exactly 2.4 GHz to be heard, which requires a precise clock source
